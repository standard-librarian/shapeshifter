# v0.1.0 Release Notes

Released: 2026-05-28.

## Highlights

- Core request and response transform engine.
- YAML/JSON spec loader for `version: "1"`.
- JSON Schema 2020-12 validation.
- jq source expression evaluation with output-count and value-size guardrails.
- Request/response field projection, validation rules, coercion, and handlers.
- Runtime and preview processing modes.
- Observer hook with no-op default.
- Echo middleware, preview API, embedded portal, and runnable Echo example.
- Gin, route-scoped Chi, and Fiber adapters.
- Static embedded contract portal with no CDN or Node build dependency.
- Language-neutral conformance fixtures and Go conformance runner.

## Stable Compatibility Surface

- `shapeshifter.yaml` `version: "1"` semantics.
- Root Go package public API listed in `docs/api-surface.md`.
- Error phases, stages, and error codes.
- Preview API JSON request/response shapes.
- Sanitized spec JSON.
- UI `config.json`.
- Conformance fixtures in `conformance/`.

## Known Limitations

- JSON object roots only.
- Buffered JSON responses only.
- No streaming, SSE, websockets, hijacking, or multipart transforms.
- Response passthrough is intentionally unsupported.
- JSON Schema `$ref` is rejected in v1.
- Route patterns are adapter-native, not normalized across frameworks.
- jq programs are trusted configuration.
- Preview/UI routes require application-owned auth.
- Try-it-out can mutate data when enabled.
